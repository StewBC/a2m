Run AppleWin and set to Color (NTSC Monitor) - Works with others too, like Color TV.
Paste the compile output into AppleWin (It will be at $6000, so set PC there and run it).  
  It will draw an image.
  Save a screenshot (PrtScr key).
  "AppleWin_ScreenShot_000000000.bmp" in my case.
The pattens from col.asm are also in patterns.txt -- exactly as .byte ....
Run the python script.
  python aw-tapper.py --image .\AppleWin_ScreenShot_000000000.bmp --patterns patterns.txt --space rgb --taps 17 --phases 8 --lam 1e-3 --emit learned_rgb_auto_bw.c
  This outputs values to learned_rgb_auto_bw.c (the --emit parameter)
Paste the contents of "learned_rgb_auto_bw.c" into the code file src\ui\nuklear\unk_apl2.c
  below // Display the double hires screen
  where it says // Auto-generated by aw-tapper.py
Now a2m renders a lot like AppleWin!
